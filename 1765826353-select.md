# SELECT

Sirve para seleccionar datos de una tabla.

Se usa de esta manera: 

```mysql
  
  SELECT `name`, `price` FROM CAR;

```

## SELECT with DISTINC 

La clausula DISTINC permite eliminar las filas duplicadas de un SELECT.
Solo elimina la fila si todas las columnas de las dos filas estan repetidas. No funciona con 
columnas individuales.

```mysql
  
  SELECT DISTINCT `name` FROM CAR;

```

## SELECT all columns (*)

```mysql
  
  SELECT * FROM stack;

```

## SELECT por nombre de columna 

```mysql

  SELECT colum_name FROM stack;

```

## SELECT with LIKE (%)

Sirve para hacer busquedas en base a un patro.

Si quiero buscar una palbra que tenga adm en cualquiera de sus partes se haría:

```mysql

  SELECT * FROM stack WHERE username LIKE "%adm%";

```

Si quiero buscar las palabras terminen en adm:

```mysql
  
  SELECT * FROM stack WHERE username LIKE "%adm";
  
```

Para buscar las palabras que empiecen con adm: 

```mysql 
  
  SELECT * FROM stack WHERE username LIKE "adm%";
  
```

Si se añade un "_" en cualquier part de una palabra a buscar, el de "_" actua como espacio en 
blanco para hacer la busqueda.

```mysql
  
  SELECT * FROM stack WHERE username LIKE "adm_n";
  
```

Buscara cualquier palabra que tenga adm seguido de cualquier letra y una n.

## CASE or IF

Crea una nueva columna en la table devuelta (no en si en la tabla de base de datos) con un atributo nuevo
cuyo valores corresponden a la condicion dada.

```mysql
  
  SELECT st.name,
  st.percentage,
  CASE WHEN st.percentage >= 35 THEN 'Pass' ELSE 'Fail' END AS `Remark`
  FROM student AS st ;

```

Result:

| name | percentage | remark |
| ---  | ---------- | ------ |
| Isha | 67         | Pass   |
| Rucha| 28         | Fail   |
| Het  | 35         | Pass   |
| Ansh | 92         | Pass   |


### If

```mysql

  SELECT st.name,
  st.percentage,
  IF(st.percentage >= 35, 'Pass', 'Fail') AS `Remark`
  FROM student AS st ;

```
## Alias

Un alias de una tabla se crea con el as.

```mysql

  SELECT username AS val FROM stack;
  SELECT username val FROM stack;

```
